=== para criar ===
```shell
cargo new validador_documentos --lib
```
=== para publicar ===
```shell
cargo login
```
=== ou ===
```shell
cargo login YOUR_API_TOKEN
```
=== para sair ===
```shell
cargo logout
```
=== para gerar o pacode de publicação ===
```shell
cargo package
```
=== para publicar ===
```shell
cargo publish
```
